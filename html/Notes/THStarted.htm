Getting Started

So you've decided to give Stars! a try, but don't know quite where to get started?

As a quick summary, the steps are:
  1. Get Stars! running
  2. Launch Stars! in your virtual environment.
  3. Select "New Game" in Stars!
  4. In the bottom left of Stars!, select "Begin Tutorial."
  5. Once you've completed the tutorial, select "New Game" again.
  6. Select "Customize Race," and create your own custom race.
  7. Select "OK," and begin your game!

Once you've got a grasp of how things work, use the TotalHost site to create a game. The steps there are:

  1. Create a TotalHost account.
  2. Upload a Race file into your profile.
  3. Create a New Game on TotalHost.
  4. Convince your friends to join and play.
  5. Launch the new game
  6. Download the .xy file from TotalHost (once)
  7. Download your .m file to the same folder
  8. Open the .m file with Stars!
  9. Save and Submit your Turn
  10. Upload the new .x file to TotalHost
  11. Await your next turn
  12. Crush them all.

    13. Welcome to Stars!
