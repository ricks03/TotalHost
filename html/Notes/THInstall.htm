Installing Stars!

Stars! is OLD. Windows 3.11 old. However, Stars! can be made to run almost anywhere.

See Stars vs OS for even more information.

Stars! on Various Operating Systems

  • Windows 16-bit or 32-bit (Windows 98, 2000, XP, Vista, 7, 8) - Stars! will just run.
  • Windows 64-bit
    • Virtual PC (no longer available from Microsoft) to install XP as a virtual machine on on Windows 7x64. Note: this does not work with Windows 7 Home as XP Mode requires Windows 7 Professional, Enterprise or Ultimate.
    • VMWare Player will let you run 32-bit Windows on a 64-bit system.
    • Oracle VirtualBox will let you run 32-bit Windows on a 64-bit system.
      • Download a working VirtualBox XP Mode VM.
    • DOSBox (Windows 7, XP, or Vista). See below.
    • StarsWine on OTVDM. Created by raptor. See below.
  • Macs
    • Wine is ported to MacOS X, and Stars! runs under Wine.
    • DosBOX runs on Mac - see below.
  • Linux: Using WINE

DOSBox:

To Run Stars! under DOSBox in 64-bit Windows:

  1. Download and install DOSBox from from http://www.dosbox.com/
  2. Download Stars! DOSBox (ZIP) or (RAR)
  3. Extract Stars! DOSBox (If you extract to C: you won't even have to edit the .conf file, otherwise, edit the included dosbox.conf to point to the current location of Stars! DOSBox)
  4. Create a shortcut on your desktop to:
    "C:\Program Files\DOSBox-0.72\dosbox.exe" -conf "C:\DOSBoxC\dosbox.conf"
    (Modify the location of DOSBox and the location of Stars! DOSBox as appropriate)
  5. Launch shortcut

StarsWine:

To Run Stars! under StarsWine in 64-bit Windows:

  1. Install StarsWine which uses OTVDM
  2. Edit the path to your OTVDM within the install.reg on two places (in the middle and at the end of the file); if you don't want his black windows use otvdmw.exe instead of otvdm.exe
  3. Double Click on the install.reg

Windows 16/32 bit

Stars! doesn't require installation. Simply download and unzip it to a directory (e.g. C:\Games\Stars!\) and launch Stars!.exe. You can create a shortcut on your Desktop. Don't forget to edit your stars.ini (created the first time Stars! is run) to set options including extended reporting, a default password and additional backups.

Basic Stars! Setup

Place the following in the [Misc] section of stars.ini:

DefaultPassword=Foo

where Foo is whatever password you generally use.

Backups=N

with N a number from 1 and 999. Backup directories will be created and stored according to the turn number, and will loop back around. By default Stars! saves one previous turn's data in a directory it creates named Backup.

newreports=1

Provides increased information in your text dumps and change the name of the dump files from gamename.pla to gamename.p## (## being the player number). Required by some utilities.

logging=1

(Version j or higher only). Only used for command line actions. Logging is ignored if the host dialog or player interface is visible.

For more information about the ini file, check out the notes on Autohost.
