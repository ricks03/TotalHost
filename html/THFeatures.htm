TotalHost Features

TotalHost (TH) is a web-based interface for Stars! game and turn-management. TH builds off the concept of AutoHost, but adds a number of features. Features include:

  • Web-based game creation (incl. player management)
  • Uploading of externally-created (zipped) games
  • Computer AI Player support (via zipped games)
  • Daily and hourly options for turn generation
  • A player-pause system, permitting managed player pauses, much like timeouts in sports
  • Configurable forced turn generation
  • Host-view mode
  • Shared player messages mode
  • Shared user-submitted news content
  • Anonymous players
  • Configurable inactive player inactivation (and setting to Housekeeping AI)
  • Download the game history (to recreate the .H file, view in retrospect, and/or recover from system failure)
  • Download messages
  • Storage and viewing of race files
  • Transfer players
  • Reset a dropped player's Stars! password
  • Providing a serial number for every user profile
  • Automatic pauses of games in case of a server Internet, reboot, or power outage
  • A client interface, to make it easier to programmatically read game status
  • Team games (where team information is automatically merged)

TotalHost can sanitize Stars file information that could be exposed to other players:

  • Mine field information
  • Mystery Trader Information
  • Worm hole information
  • CA information

TotalHost can alert on turn submission (and even fix) a number of Stars! Bugs/Exploits:

  • Prevent accidentally triggering Copy Protection
  • Cheap Colonizer/Colonization Module
  • Cheap Starbase
  • Mineral Upload
  • Space Dock Armor slot Buffer Overflow
  • Starbase Friendly Fire Battle Plan
  • Prevent Crash Stars Bug (Player 1 Fleet 1 at Last Player's planet of 10th starbase design)
  • Prevent 32k Fleet Merge
  • Prevent SS Pop Steal

Standalone Utilities

  • StarsPWD: Reset password on various files (.r, .m, .x, .hst)
  • StarsRace: Read and uncorrupt race files
  • StarsPlanet: Output all planet information
  • StarsScore: Output player scores
  • StarsAI: Change player/AI status
  • StarsMSG: Export turn file messages
  • StarsFix/StarsClean: Fix & Clean turn files manually
  • Check turns for copy protection triggers
  • StarsGraph: Create graphs of player resources/score
  • StarsMoveXY: Move planets (and associated fleets) about
  • StarsShape: Modify StarsMoveXY data to create universes in shapes (spiral, arm, etc).
  • StarsMerge: Merge .m (and .h) files
  • StarsMakeX: Create a (blank) submitted turn
  • StarsTrimM: Strip multi-turn data out of a .m file
  • Various other Perl CLI tools for analyzing and reporting on Stars! file data.
