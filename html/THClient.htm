TH has API support for a modified AutoHost Java client, ex: https://totalhost.starsautohost.org/scripts/page.pl?client=java&GameFile=testfile

Example:
game-name=Full test
short-game-name=6a7b2df9
game-year=2404
status=Active
hosted-by=test8
next-gen-time=2026-02-25 05:29:17 PST
last-gen=2026-02-23 12:10:07 PST
game-created=2026-02-22 21:26:16 PST
current-time=2026-02-23 13:11:19 PST
generate-as-available=1
player1-race=The Bulushis
player1-turn=Out Not Submitted
player2-race=The WM
player2-turn=Out Not Submitted
player3-race=The joats
player3-turn=Out Not Submitted
player4-race=The Humanoids
player4-turn=Out Not Submitted

Status: Pending Start, Pending Closed, In Progress, Delayed, Paused, Needs Replacement, Creation in Progress, AWaiting Players, Finished
Generate as available: if all turns are in, generate (0/1)

Time is sensitive to the time zone set in the user's profile if logged in
